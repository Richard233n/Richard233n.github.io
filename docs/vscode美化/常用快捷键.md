## 常用快捷键

| 快捷键                 | 功能                                               |
| ---------------------- | -------------------------------------------------- |
| `Ctrl + P`             | 打开文件                                           |
| `Ctrl + Shift + N`     | 打开新窗口                                         |
| `Ctrl +  B`            | 显示/隐藏侧边栏                                    |
| `Ctrl +  J`            | 显示/隐藏终端                                      |
| `Ctrl +  P`            | 打开命令面板                                       |
| `Ctrl +  P + @`        | 列出当前代码的大纲                                 |
| `Ctrl + Shift + F`     | 全局搜索                                           |
| `Ctrl + Shift + B`     | 切换全屏                                           |
| `Ctrl + /`             | 注释当前行                                         |
| `Ctrl + ←`             | 移动光标到行首                                     |
| `Ctrl + →`             | 移动光标到行尾                                     |
| `Ctrl + shift + enter` | 移动光标到上一新行                                 |
| `Ctrl + enter`         | 移动光标到下一新行                                 |
| `Ctrl + Shift + K`     | 删除当前行                                         |
| `Ctrl + Shift + K`     | 列出当前代码的大纲                                 |
| `Ctrl + Shift + L`     | 选中一个字符后，按当前字符完全匹配的字符将会被选中 |
| `Ctrl + D`             | 选中一个字符后，按当前字符完全匹配的字符逐个选中   |
| `Ctrl + C`             | 复制行                                             |
| `Ctrl + X`             | 剪切行                                             |
| `Ctrl + Z`             | 撤销                                               |
| `Ctrl + F`             | 查找                                               |
| `Ctrl + A`             | 选择所有                                           |
